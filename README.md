# home-loan-default-prediction


## Author

Aniket Chauhan

## Project Overview

This project focuses on predicting the likelihood of a customer defaulting on a home loan using machine learning techniques. The analysis is based on customer demographic and financial information collected from multiple datasets.

## Objectives

* Analyze customer loan and credit data.
* Identify factors influencing loan default.
* Build predictive models to assess default risk.
* Generate insights to support better lending decisions.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* XGBoost

## Project Workflow

1. Data Collection and Merging
2. Data Cleaning and Missing Value Treatment
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Preprocessing
6. Model Building and Evaluation

## Machine Learning Models Used

* Logistic Regression
* Random Forest
* XGBoost

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

## Business Impact

The model can help financial institutions identify high-risk applicants and improve loan approval decisions, thereby reducing potential credit losses.

## Conclusion

The project successfully developed predictive models to identify customers with a higher probability of default. The analysis also highlighted important factors influencing loan repayment behavior.
